# NumPy for Data Science 🧮

A comprehensive repository documenting my journey of learning **NumPy**, the fundamental library for numerical computing in Python. This repository contains concepts, examples, practice exercises, and notes to build a strong foundation for **Data Science**, **Machine Learning**, and **Scientific Computing**.

---

## 📌 Topics Covered

* Introduction to NumPy
* Creating NumPy Arrays
* Array Attributes
* Indexing & Slicing
* Reshaping Arrays
* Mathematical Operations
* Broadcasting
* Universal Functions (UFuncs)
* Aggregation Functions
* Random Number Generation
* Linear Algebra Basics
* Statistics with NumPy
* Practice Problems
